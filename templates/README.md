# Templates Folder

This folder is used to store ESL (Electronic Shelf Label) template files in YAML format.

## Purpose

When you use the "Load template" or "Save template" options in the Template Designer:
- The file picker will start in or near this directory (depending on browser support)
- Save your `.yaml` or `.yml` template files here for easy access
- All your template designs can be organized in this centralized location

## Usage

1. **Save Template**: Click File → Save template in the editor to save your template as a YAML file
2. **Load Template**: Click File → Load template to open an existing YAML template file

## File Format

Template files should be in YAML format (`.yaml` or `.yml`) and contain ESL element definitions including:
- Text elements
- Rectangle shapes
- Lines
- Barcodes
- QR codes
- Images
- Template configuration (canvas size, ESL type, etc.)
